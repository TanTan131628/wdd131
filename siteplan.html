Project Site Plan

Site Name

Name: Pack and Go Express

This name represents a courier service providing efficient and reliable package delivery.

Site Purpose

The site will provide information about courier services, pricing, tracking, and contact options.

Scenarios

  • How can I track my package in real time?
  • What are the shipping rates for international deliveries?

Color Schema

Primary Color: #0047BB Deep Blue (Used for buttons and highlights)

Secondary Color: #6A11CB Gradient Purple (Used for headers and accents)

Typography

Primary Font: Poppins (Used for body text and headings)
